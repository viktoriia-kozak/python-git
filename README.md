# python-git
 Навчальний проект з Python для роботи з Git та GitHub.
 ## Project structure
 - 'data.py'-завантаження та підготовка даних
 - 'analysis.py'- аналітичні функції
 - 'report.py'- запуск аналізу та вивід результатів
 ## Technologies
- Python
- pandas
- Git/GitHub
## How to run
1. Клонувати репозиторій
2. Запустити файл 'report.py'
